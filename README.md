# Simple Weather App

A lightweight Python-based weather application that provides current weather information and forecasts.

## Features

- **Current Weather**: Get real-time weather conditions for any location
- **Weather Forecasts**: View upcoming weather predictions
- **Simple Interface**: Easy-to-use command-line interface
- **Weather Details**: Temperature, humidity, wind speed, and more

## Installation

1. Clone the repository:
```bash
git clone https://github.com/HubertSuchorowski/Simple-Weather-App.git
cd Simple-Weather-App
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application:
```bash
python weather_app.py
```

Follow the on-screen prompts to:
- Enter a city name
- View current weather conditions
- Check weather forecasts

## Requirements

- Python 3.6+
- See `requirements.txt` for additional dependencies

## API

This app uses a weather API to fetch real-time data. Make sure you have a valid API key configured.
